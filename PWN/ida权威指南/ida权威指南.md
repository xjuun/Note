# 第一章 反汇编简介

## 1.1 x86汇编语法

### 1.1.1 AT&T

-   以`%`为所有寄存器名称的前缀，以`$`作为文字常量

-   源操作书在左边，目的操作数在右边

例如：eax加4： add $0x4,%eax

### 1.1.2 Intel

例如：eax加4： add eax,0x4
